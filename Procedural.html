The Goal

The goal of this experiment was to see if I could procedurally decorate and place hallways and cooridors as if they were placed by hand using blueprints and level instances in Unreal Engine 5.


Decoration System

  • Helper nodes are placed around nodes that specify what kind of props and decals could be placed.
  • Prop categories have a list of valid props that can be placed based on the node. A TV should never be placed on the ground for example.

Generation System

  • The generator picks a starting room and creates a path. This path continues until a specified amount of rooms generated is reached.
  • When a room is placed it checks if it is colliding with another, if so it is deleted and a new room is placed. If this fails multiple times the end is closed and a new continuation point is picked.
  • After the end room is placed, the rest of the openings are filled with random rooms or closed off. This continues until all rooms are closed off.
