Sky Jellies

An experimental unity game about exploring and taking care of creatures called Jellies. This prototype was created by P1-OC, and I had a hand in assisting in development along with many other developers.

Item Tossing

  • Using already created inventory systems, I developed a way for players to drop items in their inventory.
  • Developed an item removal system to ensure that items would get removed from the inventory when tossed.
  • Integrated the item collection system so that when dropped items were walked over, the same item and amount would be returned to the player.

Player Rigidbody

  • Transformed the CharacterController into RigidBody to improve player movement to make physics feel less floaty and more responsive.
  • Reworked the jump systems to allow for coyote time.
  • Additionally added a system that would detect if the jump input was pressed while airborn to allow for a delayed jump if the player was going to hit the ground within a certain time.

Item Holding

  • Created a hotbar so that players could drag items from their inventory to the hotbar in order to interact with the environment.
  • Pressing the number keys 1,2, and 3 would select items within that slot.
  • Item model and texture automatically display when the item is held in the hotbar.

Block placement

  • Developed a grid system that detects ground collisions and other blocks to prevent overlap.
  • Blocks use the inventory and hotbar systems, and blocks subtract when placed.

Item Interaction

  • Rather than dragging and dropping items onto interactables, players now can have a selected item on their hotbar and press the interact key to directly interact with these items.
  • For Jellies, players can feed them by holding food out in their hand and interacting with them.
  • Additionally, players can also pet Jellies by having nothing equipped and interacting with them.
